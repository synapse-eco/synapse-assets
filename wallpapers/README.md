# Synapse Wallpapers

This directory contains the official wallpapers of the Synapse Project.

These wallpapers are designed to provide a consistent visual experience across the Synapse ecosystem and represent the project's official artwork for desktops, login screens, promotional materials, and other approved uses.

---

## Directory Structure

* **png/** — High-quality wallpapers for general use.
* **bmp/** — Bitmap versions for compatibility where required.
* **source/** — Original editable project files used to create and maintain the wallpapers.

---

## Usage

The wallpapers in this directory are intended for use in official Synapse environments and materials.

Whenever possible, use the original exported assets without modification to preserve the intended visual quality and consistency.

The editable source files are provided solely for the maintenance and future development of the official wallpaper collection.

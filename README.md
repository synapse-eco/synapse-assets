# Synapse Assets

The **Synapse Assets** repository is the official source of visual identity assets for the Synapse ecosystem.

It provides a structured, version-controlled collection of graphical resources used across the Synapse operating system, tools, documentation, and official publications.

---

## Purpose

This repository exists to:

* Maintain the official Synapse visual identity.
* Store and version all official logos and wallpapers.
* Provide consistent asset formats for different platforms and use cases.
* Preserve editable source files for long-term development.
* Serve as the authoritative source for Synapse branding materials.

---

## Repository Structure

The repository is organized by asset type and format:

```text
logos/        → Official Synapse logos and branding marks
wallpapers/   → Official Synapse wallpapers and background artwork
```

Each category contains standardized subdirectories:

* `png/` — General-purpose raster images.
* `svg/` — Scalable vector assets for UI, design, and high-resolution use.
* `bmp/` — Compatibility formats for legacy or low-level systems.
* `source/` — Editable original project files used to create assets.

---

## Logos

The `logos/` directory contains all official Synapse branding marks, including primary logos, text-based logos, and related identity assets.

These assets represent the core visual identity of Synapse and are used across the ecosystem.

See `logos/README.md` for detailed usage information.

---

## Wallpapers

The `wallpapers/` directory contains official Synapse background artwork designed for desktops, lock screens, and promotional materials.

These wallpapers are part of the Synapse visual experience and help establish a consistent aesthetic across the system.

See `wallpapers/README.md` for detailed usage information.

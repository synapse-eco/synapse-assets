# Synapse Logos

This directory contains the official logos and logo-related assets of the Synapse Project.

The contents of this directory serve as the authoritative source for the project's visual identity and are maintained in multiple formats to support development, documentation, websites, applications, and other official Synapse resources.

---

## Directory Structure

* **svg/** — Scalable Vector Graphics for high-quality, resolution-independent use.
* **png/** — Portable Network Graphics for general-purpose use.
* **bmp/** — Bitmap images for compatibility where required.
* **source/** — Original editable project files used to create and maintain the official logos.

---

## Usage

Use the most appropriate format for your intended purpose:

| Format     | Recommended Use                                                 |
| ---------- | --------------------------------------------------------------- |
| **SVG**    | Official branding, UI, printing, and scalable graphics          |
| **PNG**    | Documentation, websites, presentations, and general digital use |
| **BMP**    | Legacy compatibility and platform-specific requirements         |
| **Source** | Editing, maintenance, and future development                    |
